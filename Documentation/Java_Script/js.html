Variable

⇨ Variable is a one type of cointner that store the value.

⇨ In JavaScript , Three type of Variable

  • Var(Local Scope)
  • Let(Global Scope)
  • Const(Block Scope)

Var-In JavaScript ,most commenly used variable.It can be Redeclare and Reasseigned values.

↳

var a = 140; // variable a can be reassigned a new value and also redeclared

Let-In JavaScript ,this variable only reassigned value and not Redeclare value.

↳

let a = 202; // variable a can't be redeclared but can be reassigned a new value.

Const-In JavaScript ,this variable is not Redeclare and reassigne and this is a keyword.

You can also use in your code in one time.

↳

const a = 5; // variable a cannot be reassigned a new value or redeclared.

if-statement-When if condition true the associated code is executed.

↳

//your code

}

else-if-statement-The else if statement hold multiple condition.
