What is jQuery?

JQuery is a small,fast and feature-rich Java Script Library.

Uses JQuery you makes things like manipulation,Event handling,animation and ajax much simpler with an easy to use APi that works across multiple browser.

jQuery has changed the way that millions of people write JavaScript.

It's a very well written code which is about 32kb size

Why use Jquery

JQuery perform lot of common tasks that require many line of javascriot code to accomplish and wraps them into methods that you can call with single line of code .

JQuery function completely browser independent

Most of biggest compaines use JQuerys such as GOOGLE,MICROSOFT,NETFLIX,IBM etc....

Syntax

$(selector).action()

  • $-indicates the access JQuery
  • selector- find HTML elemnts
  • action- action performed on the elements

Common selector

  • * - select all elements
  • # - select element with id name
  • . - select element with class name and you can multiple class
  • select element with tag name and you use multiple tag name
