What is jQuery?

↠jQuery is a small, light-weight and fast JavaScript library

↠It is cross-platform and supports different types of browsers.

↠It is also referred as ?write less do more? because it takes a lot of common tasks that requires many lines of JavaScript code to accomplish, and binds them into methods that can be called with a single line of code whenever needed.

↠It is also very useful to simplify a lot of the complicated things from JavaScript, like AJAX calls and DOM manipulation.

Features

↳
  • HTML manipulation
  • DOM manipulation
  • CSS manipulation
  • JSON Parsing

jQuery Animation

↠jQuery provides various methods for creating animations. One common method is animate().

Syntax

$("selector").animate({params}, speed, callback);

Example

↳

<style>

#myElement {

width: 100px;

height: 100px;

background-color: red;

}

</style>



<div> id="myElement"></div>



<script>

$('#myElement').animate({

width: '200px',

height: '200px', });

backgroundColor: 'blue'

}, 1000);

});

</script>
